Persistent serving_tuning history lives under this tree.

Expected layout:

- `serving_tuning/latest/row_state.json`
  Latest per-row state used for staleness checks and rerun decisions.
- `serving_tuning/runs/full/`
  Immutable full per-build snapshots of compiled row results.
- `serving_tuning/runs/best/`
  Immutable best-per-row snapshots produced during workbook writeback.

These files are generated by the Jenkins/automation flow. The manifest workbook
remains under `automation/manifests/`.
